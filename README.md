# ZenCity Paris

Application de découverte des lieux zen et restaurants à Paris.

## Fonctionnalités

- 🌿 Découverte de spots zen
- 🍽️ Guide des restaurants
- 📍 Carte interactive
- ⭐ Système d'avis
- 🎮 Gamification

## Installation

```bash
git clone https://github.com/CodeKiShow/zencity-paris.git
cd zencity-paris
npm install
npm start
```
